# Runbooks

Operational procedures for common tasks and incident response.

## Template

Each runbook covers:
- **Trigger** — when to use this runbook
- **Prerequisites** — access, tools needed
- **Steps** — numbered, copy-pasteable commands
- **Rollback** — how to undo
